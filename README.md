# Todo List App

## Project made using JavaScript, TypeScript, React and Redux.

#### APP FEATURES:

- Loading data from JSONPlaceholder
- Adding todo
- Deleting todo
- Editing todo
- Pagination

---

#### USAGE

```
git clone https://github.com/Alejandro-Vas/todo-list-app.git
cd todo-list-app
npm i
npm start

```
